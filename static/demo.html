Facebook Page Widget

Embed recent posts from your Facebook page with a single <script> tag.

Embed Code


    


    

      
Parameters

      

        

          
data-page-id

          
Facebook page ID to fetch posts from (required)

        

        

          
data-limit

          
Number of posts to fetch, 1–20 (default: 5)

        

        

          
data-skin

          
light, dark, or vivid (default: light)

        

        

          
data-color-bg

          
Override the background color. Accepts any CSS color value (hex, rgb, hsl, named).

        

        

          
data-color-text

          
Override the primary text color.

        

        

          
data-color-accent

          
Override the accent / Follow button color.

        

        

          
data-only-keywords

          
Comma-separated keywords — only posts containing at least one of these are shown. E.g. event,konsert

        

        

          
data-skip-keywords

          
Comma-separated keywords — posts containing any of these are hidden. E.g. menu,meny,specials
